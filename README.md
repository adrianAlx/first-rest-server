# REST Server

My first Basic REST Server.

Technologies:

- Node.js
- Express
- Mongoose

Postman Docs:
[**API Documentation**](https://documenter.getpostman.com/view/15552178/UVJYJeQL#213208fd-1eb9-4bcd-a174-e169159f211e)

Rebuild node modules:

```bash
npm i
```

Run with Nodemon:

```bash
npm run dev
```
